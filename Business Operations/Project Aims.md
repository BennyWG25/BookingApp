Project Title: Mad Swans Booking Prototype
Aims: Design and build a prototype booking solution for a multi-sport country club (Mad Swans).

Scope:
- To produce a design for full web app and technical feasibility of features.
- To build a demo site with the minimum viable product.
- To present a clear business proposal for our solution.

Objectives:
1. Build an interactive web app using licensed packages with core features which can be demoed by April 30th.
2. Develop a business strategy and product development timeline by April 1st to demonstrate to customers what further features they can expect - necessary for investment. 
## WP1 : Client needs and solutions
Aim: To produce a planned feature list, with key features for the minimum viable product identified.
